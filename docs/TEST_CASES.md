# Задание 1: Тест-кейсы для выпадающего списка стран

## Условие
В веб-приложении есть выпадающий список с выбором стран и кодом телефона. Необходимо проверить корректность работы списка.

## Предположения
- При выборе страны автоматически подставляется соответствующий код телефона
- Список стран отсортирован в алфавитном порядке
- Поле выбора страны обязательно для заполнения

## Позитивные тест-кейсы

| ID | Название | Шаги | Ожидаемый результат |
|----|----------|------|-------------------|
| P1 | Отображение списка по умолчанию | 1. Открыть страницу с формой<br>2. Найти выпадающий список стран | Список свернут. Отображается значение по умолчанию |
| P2 | Раскрытие списка стран | 1. Кликнуть на выпадающий список | Список раскрывается, отображая полный перечень стран |
| P3 | Выбор страны и отображение кода телефона | 1. Раскрыть список<br>2. Выбрать "Франция" | В поле страны отображается "Франция", в поле кода телефона "+33" |
| P4 | Последовательный выбор разных стран | 1. Выбрать "Япония"<br>2. Выбрать "Бразилия" | Код телефона меняется с "+81" на "+55" соответственно |
| P5 | Проверка сортировки списка | 1. Раскрыть список стран | Страны отсортированы в алфавитном порядке |

## Негативные тест-кейсы

| ID | Название | Шаги | Ожидаемый результат |
|----|----------|------|-------------------|
| N1 | Отправка формы без выбора страны | 1. Оставить выбор страны по умолчанию<br>2. Заполнить другие поля<br>3. Нажать "Отправить" | Форма не отправляется, поле подсвечивается с ошибкой |
| N2 | Ввод некорректных данных в поле телефона | 1. Выбрать страну<br>2. Ввести буквы в поле телефона | Буквы не вводятся или отображается ошибка формата |
| N3 | Быстрый последовательный выбор | 1. Быстро выбрать 5-10 разных стран подряд | Система стабильно обрабатывает запросы, код соответствует последней выбранной стране |

## Критические сценарии
- Проверка отображения стран с длинными названиями
- Проверка работы при медленном соединении
- Проверка поддержки специальных символов в названиях стран
